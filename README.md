# Métodos Numéricos - Sistemas de Ecuaciones Lineales

App interactiva en Streamlit que muestra el paso a paso de distintos algoritmos para resolver sistemas de ecuaciones lineales.

## Algoritmos incluidos

- **Factorización LU** (Método Directo)
- **Jacobi** (Método Iterativo)
- **Gauss-Seidel** (Método Iterativo)
- **SOR** - Sobrerelajación Sucesiva (Método Avanzado)
- **Gradiente Conjugado Precondicionado** (Método Avanzado)

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy en Streamlit Community Cloud

1. Sube este repositorio a GitHub.
2. Ve a [share.streamlit.io](https://share.streamlit.io).
3. Conecta tu repo y apunta al archivo `app.py`.
